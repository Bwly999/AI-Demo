🎨 HSV 颜色空间

HSV 是更符合人直觉的颜色模型：
H (色调) = 颜色种类   S (饱和度) = 鲜艳程度   V (明度) = 亮度

H: {{ hsv.h.toFixed(2) }}
S: {{ hsv.s.toFixed(2) }}
V: {{ hsv.v.toFixed(2) }}
🌈 H (色调) {{ (hsv.h * 360).toFixed(0) }}°
💧 S (饱和度) {{ (hsv.s * 100).toFixed(0) }}%
☀️ V (明度) {{ (hsv.v * 100).toFixed(0) }}%
📖 HSV → RGB 转换
C = V × S
X = C × (1 - |(H×6) mod 2 - 1|)
m = V - C
(R',G',B') = 根据H扇区查表
(R,G,B) = (R'+m, G'+m, B'+m) × 255
⬛ 灰度转换

灰度值通过加权平均计算，绿色权重最高是因为人眼对绿光最敏感。

→
灰度值: {{ grayValue }}
Gray = 0.299·R + 0.587·G + 0.114·B
= 0.299×{{ rgb.r }} + 0.587×{{ rgb.g }} + 0.114×{{ rgb.b }}
≈ {{ grayValue }}
