# 手机产线尺寸相关性分析（教学项目）

> 一个用 **Jupyter Notebook** 讲解"手机产线组装前后尺寸关联性 + 不良品因子分析"的完整教学案例。
> 数据全部在 Notebook 内用种子随机数生成，无需任何外部文件，**可复现**。

## 适用人群

- 想系统学习"数据相关性分析"全部主流方法的工程师 / 分析师
- 制造业（尤其手机 / 消费电子组装产线）质量工程师
- 用真实产线数据做尺寸关联分析前的"练手沙盘"

|---|---|---|
| Pearson / Spearman / Kendall 相关 | 连续变量两两是否相关 | 第 3 章 |

## 项目特色：数据里"埋"了一套潜在工艺结构

为了让分析"有东西可挖"，500×200 的数据集由 **6 个不可观测的潜在工艺因子** 驱动生成：

| 因子 | 工艺含义 | 对不良的影响 |
|---|---|---|
| F1 | 点胶密封 | **强相关** |
| F2 | 屏幕贴合 | **强相关** |
| F3 | 中框冲压 | 中等相关 |
| F4 | 电池装配 | 弱相关 |
| F5 | 螺丝锁附扭矩 | 噪声（无关） |
| F6 | 环境温湿度 | 噪声（无关） |

200 个尺寸都是这 6 个因子的线性组合 + 噪声 → 自然产生**多重共线性**；
不良率由 `sigmoid(β0 + 2.0·F1 + 1.8·F2 + 0.6·F3)` 决定 → 让统计/ML 方法都能"挖"出来。

> 🎯 教学闭环：你能通过分析逐步"恢复"出这 6 个因子，并与生成真相对照验证。

## 数据集结构（500 行 × 200 列）

- `X_pre_001` ~ `X_pre_100`：组装前零件尺寸（100 列）
- `X_post_001` ~ `X_post_099`：组装后整机尺寸（99 列）
- `label`：不良品标记 `0/1`（不良率约 8–12%）
